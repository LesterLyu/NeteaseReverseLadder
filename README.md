# NeteaseReverseLadder
Play your favorite songs in Netease Cloud Music that are banned outside Mainland China. 
解除网易云音乐海外播放限制。
Test ip from http://cn-proxy.com/
# 下载
https://github.com/LesterLyu/NeteaseReverseLadder/releases

网易云和qq音乐里 设置http代理
- 地址： 127.0.0.1
- 端口： 15213
